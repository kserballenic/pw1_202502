Concepto de Tabla

La tabla es una forma de organizar datos en forma tabular para mostrar la información pertinente a un entidad. La data se organiza en celdas que intercectan entre filas y columnas para demostrar una relación. HTML permite la representación de datos en una estructura de elementos HTML anidados que definen la estructura y relación de la data a presentar.
